## 幻方在经历了年初DMA爆仓和9月空头巨亏后，宣布不再做中性策略，转型纯多头指数增强。这个我看不懂，什么叫做“中性策略”，什么叫做“多头指数增强”？

这两个概念其实不复杂，我用一个比喻来帮你理解。

**先说"中性策略"（市场中性策略）**

假设你觉得自己的选股能力很强——你选的股票能跑赢大盘。但你担心一个问题：如果整个市场暴跌怎么办？再好的股票在大盘崩盘时也会跌。

中性策略的做法就是"两头下注"：

- **做多端**：买入你精选的一篮子股票（比如100只你看好的）
- **做空端**：同时做空股指期货（比如卖空中证500期货）

这样做的效果是：大盘涨跌对你影响不大了。大盘跌的时候，你股票端亏钱，但做空端赚钱，两边对冲。你真正赚的是"选股超额收益"——你的股票比指数跌得少、涨得多，那个差值就是你的利润。

打个比方：你赌的不是"谁赢"，而是"你的马比别人的马跑得快"。赛道是上坡还是下坡无所谓，只要你的马相对更快就行。

**幻方2024年为什么在中性策略上亏惨了？**

第一次是年初：幻方的做多端买了大量小微盘股，结果小微盘股遭遇踩踏式暴跌（跌停卖不出去），而做空端的股指期货没跌那么多——两边没有对冲住，直接爆仓。

第二次是9月：A股突然暴涨（政策利好），做空端的股指期货大幅亏损（你做空的东西在涨），而做多端的小盘股涨幅又没期货涨得多——又是两头亏。

两次打击之后，幻方决定放弃做空端，转型纯多头。

**再说"多头指数增强"（指数增强策略）**

这个就简单多了：

- **只做多**：不买空头，不做对冲，就老老实实买股票
- **跟踪某个指数**：比如跟踪中证500，保证持仓的大方向和指数一致
- **"增强"部分**：在跟踪指数的基础上，用量化模型调整权重——看好的股票多买一点，不看好的少买一点，争取比指数多赚一些

打个比方：指数增强就像"抄作业但做了一些自己的修改"。中证500里500只股票每只权重0.2%，量化模型觉得A股票好就加到0.5%，觉得B股票差就降到0.05%。最终目标是跑赢指数本身。

**幻方转型前后对比：**

|              | 以前（中性策略）               | 现在（多头指增）             |
| ------------ | ------------------------------ | ---------------------------- |
| 市场跌的时候 | 理论上不亏（多空对冲）         | 会跟着跌，但争取比指数跌得少 |
| 市场涨的时候 | 赚得不多（对冲削掉了部分收益） | 跟着涨，还争取比指数涨得多   |
| 极端行情     | 可能两头亏（2024年教训）       | 只承受单边风险（跌就跌）     |
| 收益特征     | 稳定但天花板低                 | 波动大但天花板高             |

所以幻方这次转型，本质上是承认了一件事：**在A股做空端的摩擦成本和极端风险太大了，不如干脆只赚"选股超额+市场上涨"两份钱。** 这也解释了为什么2025年牛市里幻方能拿到+56%的收益——纯多头策略在上涨市里天然比对冲策略赚得多。

不过反过来说，如果未来A股再来一轮大跌，幻方的多头指增产品也会跟着跌，只是争取比指数跌得少一些。这一点和国金量化多因子是一样的——大家都是纯多头，市场跌的时候谁都跑不掉。
