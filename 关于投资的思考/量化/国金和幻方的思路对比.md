# 国金和幻方的核心理念有什么不同？

相比幻方的"黑箱"，马芳接受过不少采访，策略思路透露得更多。我帮你梳理一下她的核心框架。

**马芳策略的核心理念可以概括为三句话：模型决定一切、持仓足够分散、收益三元归因。**

---

**第一，"模型决定一切"**

国金量化的所有买卖决策完全由模型信号驱动，排除人为干预。马芳自己说过："我不认为人工的干预会带来一个长期稳定的阿尔法。" 这意味着不存在"基金经理拍脑袋决定加仓某只股票"的情况——模型说什么就是什么。

不过她也不是完全"闭眼信模型"，团队会用主观监控来管理禁投池（比如财务造假嫌疑、流动性极差的股票会被排除），但具体买哪只、买多少、什么时候卖，全部是模型说了算。

**第二，"收益三元归因"——这是理解她策略最关键的一把钥匙**

马芳把超额收益拆成三块来源：

| 来源     | 占比  | 含义                                       |
| -------- | ----- | ------------------------------------------ |
| 个股超额 | 约1/3 | 模型选出的个股比同行业其他股票表现更好     |
| 行业偏离 | 约1/3 | 模型判断某些行业未来会有超额，主动超配     |
| 风格偏离 | 约1/3 | 模型在大小盘、价值成长等风格上主动偏离基准 |

这和很多传统量化基金不同——传统公募量化主要靠"个股超额"（在指数成分股里选更好的股票），行业和风格尽量贴近基准。但马芳的框架里，行业判断和风格择时各贡献了1/3的收益，加起来占了2/3。

举个例子：如果模型判断接下来AI算力产业链会有超额，就会主动增配相关行业的股票；如果模型判断小盘股相对大盘股更有性价比，就会在市值风格上向小盘倾斜。这不是马芳"主观看好AI"或"主观看好小盘"，而是模型根据历史数据训练出来的预测信号。

**第三，"持仓足够分散"**

国金量化多因子同时持有约**700只股票**，前十大重仓股合计占比仅约11%。权益仓位常年保持在近90%的高位，不做择时（不会因为觉得市场要跌就减仓）。

这个分散程度意味着：即使某只股票踩雷，对整体净值的影响也微乎其微。它赚的不是"押中一只牛股"的钱，而是"700只股票里55%跑赢、45%跑输"的大数定律。

**第四，因子挖掘和AI应用**

马芳团队用机器学习模型来做三件事：

- **预测个股超额**：输入量价数据、基本面数据、行业数据，训练模型预测每只股票未来的相对收益
- **预测行业超额**：判断哪些行业未来一段时间会有超额收益
- **预测风格确定性**：判断当前环境下，偏大盘还是偏小盘、偏价值还是偏成长更有利

和幻方的区别在于：幻方是"纯AI自动挖掘因子"（让神经网络自己发现规律），马芳更偏"机器学习辅助传统因子框架"——团队会先定义好因子大类（动量、价值、质量等），再用机器学习来决定各个因子在不同市场环境下的权重，以及哪些个股在这些因子上得分更高。

**第五，中低频交易，定位策略容量**

马芳明确说过，团队"一直在中低频的赛道上去做"。2025年换手率约881%（双边约8.8倍/年），平均到每个交易日调仓比例很小。这个频率的好处是策略容量大（40亿规模仍然能跑出超额），交易成本可控。

她也提到公募做不了日内回转交易（T+1限制），所以高频策略在公募里本身就行不通，中低频反而是公募量化的最优解。

---

**和幻方的对比总结：**

| 维度       | 国金量化（马芳）         | 幻方量化（梁文锋）            |
| ---------- | ------------------------ | ----------------------------- |
| 因子来源   | 传统因子框架+ML优化权重  | 纯AI自动发现因子              |
| 收益来源   | 个股1/3+行业1/3+风格1/3  | 未披露（推测以个股alpha为主） |
| 持仓数量   | ~700只                   | 更多（可能上千只）            |
| 换手率     | ~881%/年（中低频）       | 中低频为主（未披露具体）      |
| 算力       | 基金公司级别             | 万卡GPU集群（碾压级）         |
| 行业判断   | 模型主动做行业偏离       | 不确定（可能不做或做得少）    |
| 策略透明度 | 相对透明（多次公开访谈） | 黑箱（从不披露）              |

简单说，马芳的思路是"**广度投资+三元归因+中低频**"——不靠深度研究某一家公司，而是靠模型在全市场700只股票上各赚一点点超额，同时在行业配置和风格选择上主动偏离来增厚收益。她更像是一个"体系化的工程师"，而不是"押注型选手"。

**风险提示：** 2024年初的最大回撤-33.79%说明，当市场风格极端切换（微盘股暴跌）时，模型的风格偏离判断也会出现偏差。马芳团队在那之后升级了风险模型来增强策略对不同环境的适应能力，但极端行情下的防御能力仍然是公募量化的共性短板。

# 你在刚才的回答中说的alpha是什么意思？

Alpha（阿尔法，符号α）是投资里一个很基础的概念，用一句话解释就是：**你的收益里，"凭本事多赚的那部分"。**

举个具体例子来说明：

假设今年中证500指数涨了20%，你买了一只基金也涨了28%。那这28%可以拆成两块：

- **Beta（贝塔）**：20%——这是"跟着大盘涨的"，你随便买个指数基金也能拿到，不算本事
- **Alpha（阿尔法）**：8%——这是"超出大盘的部分"，是基金经理靠选股、择时等能力多赚出来的

所以当我们说马芳的"个股超额收益"或幻方的"超额收益"时，指的就是alpha——比基准指数多赚的那部分。

反过来，如果一只基金今年涨了15%，但指数涨了20%，那alpha就是-5%，说明基金经理不但没跑赢市场，还拖了后腿。

为什么alpha对量化基金特别重要？因为量化基金收取管理费的理由就是"我能给你跑出alpha"。如果alpha为零，投资者还不如直接买便宜的指数基金（管理费只有0.1%-0.5%），何必付1.2%的管理费给量化基金呢？所以alpha是量化基金存在的核心价值，也是评价一个量化基金经理"真实水平"最关键的指标。
