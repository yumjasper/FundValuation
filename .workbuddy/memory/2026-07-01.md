## 基金估值实时计算网页
- 交付文件:`fund-valuation.html`(单文件纯前端应用)。
- 需求:按代码添加基金到分组,基于每季度最新公布持仓比例做实时估值。
- 技术方案:全部用公开接口 + script 注入(JSONP 思路)绕过跨域,**无需注册任何 API**。
  - 基金实时估值:`https://fundgz.1234567.com.cn/js/{code}.js`(回调 jsonpgz,官方估值)
  - 季度前十大重仓股:`https://fundf10.eastmoney.com/FundArchivesDatas.aspx?type=jjcc&code={code}&topline=10`(返回 var apidata={content:HTML},DOMParser 解析)
  - 股票实时行情:`https://qt.gtimg.cn/q=sh600519,...`(GBK,价格/涨跌幅是数字不受乱码影响;第4/5/33字段;secid 规则 6→sh 0/3→sz 4/8→bj)
- 自算估值 = Σ(重仓股占净值比例 × 该股实时涨跌幅),显示覆盖度;同时展示官方估值做对比。
- 数据本地 localStorage 持久化,30s 自动刷新。
- 备选方案(若接口失效):新浪行情 hq.sinajs.cn(需 Referer,file:// 下受限);基金持仓可换 fund.eastmoney.com 的 pingzhongdata。
- 涨红跌绿(A股惯例)。
